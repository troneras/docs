### Add your credentials to the installer

#### Configure the installe

Our Installer automatically creates the resources in your AWS and Cloudflare
account. You need to configure the installer with your AWS and Cloudflare
credentials.

1. Extract `selfhoster-pro.zip` in your local machine.
2. Copy `secrets.auto.tfvars.example` to `secrets.auto.tfvars` and update the file with your credentials.
3. Copy `variables.auto.tfvars.example` to `variables.auto.tfvars` and update the file with your domain and subdomain.
4. Add your public ssh key to `terraform/config/ssh/public_keys` file. (copy from `~/.ssh/id_rsa.pub` or generate a new one)

You can leave the rest of defaults as they are. If you modify the instance type, make sure to also update the `ami_id` in the `variables.auto.tfvars` file.

### Run the installer

Run the installer `./selfhoster --init` in the root of the project.
The installer will create the resources in your AWS account and configure the DNS records in Cloudflare.

## Access coolify

If you have followed the steps correctly, you should be able to access coolify dashboard at the following URL: `https://ops.yourdomain.com`
If it's the first time accessing the dashboard, you will be asked to authenticate by cloudflare.
